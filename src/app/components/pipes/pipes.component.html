pipes works!

In Angular, a Pipe is a way to transform data in templates. Pipes allow you to format data directly in the HTML view without modifying the component class logic.

{{ 0.25 | percent }}
{{ 'hello' | uppercase }}
{{ 1234.5 | currency:'USD' }}
{{ [1,2,3,4,5] | slice:1:4 }}
{{ gender | i18nSelect: genderMap }}
